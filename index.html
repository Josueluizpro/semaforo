Sobre o Projeto

Aproveitando uma tábua que seria descartada, nossos alunos criaram uma maquete representando uma rua com iluminação e semáforo automatizados. A base foi pintada de cinza para simbolizar o asfalto, com faixas amarelas no centro e faixas de pedestre brancas. Foram usados canudos de pirulito para montar postes de iluminação e semáforos com LEDs, programados via Arduino.
